Custom Site Selectors

Add your own site selectors for form filling. These will be used instead of the built-in ones when matched.

Available Field Types Reference

- input: Regular input field - inputShadow: Input within shadow DOM - checkCheckbox: Checkbox to check - simpleClick: Element to click - dispatchedClick: Element to click using dispatched event - multiStep: Multiple operations in sequence
