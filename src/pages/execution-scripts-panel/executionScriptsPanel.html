Scripts cannot be executed on chrome:// or chrome-extension:// pages

Execution Scripts are one-time scripts that run immediately when executed. Unlike User Scripts, they don't persist on the page and don't create buttons. Use these for quick, one-off operations.

Note: Some websites may block these scripts due to Content Security Policy (CSP) restrictions.
